# 🎬 Editkaro.in Portfolio Website

A modern, responsive portfolio website for **Editkaro.in**, a video editing and social media marketing agency. The website showcases different editing categories, an interactive portfolio gallery with video previews, smooth animations, and a clean user interface.

## 🌐 Live Demo

> Add your deployed website link here after deployment.

Example:
```
https://chitranshu-dtu-official.github.io/Editkaro-Portfolio/
```

---

## 📌 Features

- 🎨 Modern and responsive UI
- 📱 Mobile-friendly design
- 🎥 Interactive video portfolio
- 🗂 Category-based portfolio filtering
- ▶ Video popup using YouTube embeds
- ✨ Smooth scroll animations (AOS)
- 📊 Animated statistics counters
- 📩 Contact form
- ⬆ Scroll-to-top button
- 🍔 Responsive navigation menu
- 🖱 Hover effects and transitions

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- AOS (Animate On Scroll)
- Font Awesome
- Google Fonts
